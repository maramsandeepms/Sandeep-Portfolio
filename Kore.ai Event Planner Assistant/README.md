# 🤖 Kore.ai Event Planner Assistant

## 📌 Project Overview
The Event Planner Assistant is a conversational AI chatbot built using the Kore.ai platform. It allows users to schedule, retrieve, and modify event details using natural language interactions.

This project demonstrates end-to-end chatbot development including dialog design, API integration, entity handling, and conversational AI orchestration.

---

## 🚀 Features
- 📅 Schedule new events
- 🔍 Retrieve event details using contact number
- ✏️ Modify existing events
- 🤖 Conversational AI using DialogGPT
- 🤖 Saerch AI for Documentaion
- 📊 Batch testing for validation
- 🔗 API integration with REST endpoints

---

## 🛠️ Tech Stack
- Kore.ai XO Platform
- Dialog Tasks & Entity Nodes
- Kore Ai featured Gen AI functionality
- Service nodes to connect to REST API
- Script nodes
- Message Node
- DialogGPT (AI-based intent handling)
- REST APIs (Mock API for event data)
- CSV (Batch Testing)

---

## ⚙️ How It Works
1. User interacts with chatbot using natural language
2. Entities extract key details (contact number, event date, etc.)
3. Dialog tasks process user intent
4. API calls fetch/store event data
5. DialogGPT enhances conversational experience
